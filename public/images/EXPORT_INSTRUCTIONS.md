# Logo Export Instructions

## Footer (Section 10/11) – Confluxe Logo

The Footer uses the Confluxe logo from Figma node **367:338** (CONFLUXE group).

**To export from Figma:**
1. Select the CONFLUXE group (or Frame containing the letter vectors)
2. Right-click → Export
3. Choose **SVG** (preferred) or **PNG** at 2x
4. Save as `public/images/confluxe-logo.svg` (or `.png`)

The current placeholder uses a text-based SVG. Replace with the exported Figma logo for pixel-perfect match.

**How to export in Figma:**
1. Open the Components page in your Figma file.
2. Select each component (right‑click → "Copy link" to verify the node ID in the URL).
3. Right‑click → Export → PNG (1x) → Export.
4. Save each file into this folder: `public/images/capabilities/` with the exact names above.

# Our Values Icon PNGs – Export Instructions

Export these 4 icon components from your Figma design (Section 7 – Our Values) as **PNG** at **1x** or **2x** scale:

| File name       | Value / Description                    |
|-----------------|----------------------------------------|
| value-icon-1.png | We take responsibility (3×3 grid: center filled, 8 outlined) |
| value-icon-2.png | We build with discipline (3×4 grid, top-left 3×3 bolder) |
| value-icon-3.png | Tech-first from day one (diamond starburst pattern) |
| value-icon-4.png | We think in decades (step/bar-chart ascending squares) |

**Where to save:**
```
public/images/values/
```

**Full paths:**
- `public/images/values/value-icon-1.png`
- `public/images/values/value-icon-2.png`
- `public/images/values/value-icon-3.png`
- `public/images/values/value-icon-4.png`

**How to export in Figma:**
1. Open Section 7 (Our Values) in your Figma file.
2. Select each icon (the layer name may be something like "Background Container" or the icon frame).
3. Right-click → Export → PNG (1x or 2x for sharpness) → Export.
4. Save each file into `public/images/values/` with the exact names above.
